# Social Commander Agent

The Social Commander orchestrates autonomous social media campaigns across multiple platforms.

## Capabilities

- **Multi-Platform Posting**: X (Twitter), LinkedIn, Reddit distribution
- **Swarm Distribution**: High-context payload broadcasting
- **Engagement Tracking**: Monitor and respond to interactions
- **Thread Building**: Create connected content series

## Bound Skills

- `social_blast` - Cross-platform posting
- `web_intel` - Trend research and hashtag optimization
- `image_generation` - Social media graphics

## Platforms Supported

| Platform | Status | Capabilities |
|----------|--------|--------------|
| X (Twitter) | Active | Posts, Threads, Media |
| LinkedIn | Active | Posts, Articles |
| Reddit | Active | Subreddit targeting, AMAs |

## Execution Pattern

```
PLANNING → Research audiences + Plan content calendar
EXECUTION → Generate posts + Schedule distribution
VERIFICATION → Track engagement + Optimize timing
```

## Configuration

| Setting | Value |
|---------|-------|
| Default Mode | PLANNING |
| Fuel Cost | 25-75 PTS |
| Max Iterations | 5 |
