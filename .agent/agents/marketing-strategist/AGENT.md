# Marketing Strategist Agent

The Marketing Strategist is a specialized agent for creating and executing comprehensive marketing campaigns.

## Capabilities

- **Campaign Planning**: Create multi-channel marketing strategies
- **Content Strategy**: Generate content calendars and messaging frameworks  
- **Competitive Analysis**: Research competitor positioning using web intel
- **Growth Metrics**: Define KPIs and success criteria

## Bound Skills

- `web_intel` - Research and competitive analysis
- `image_generation` - Campaign visual assets
- `social_blast` - Distribution execution

## Execution Pattern

```
PLANNING → Analyze market + Define strategy
EXECUTION → Generate assets + Schedule distribution  
VERIFICATION → Validate reach + Measure engagement
```

## Configuration

| Setting | Value |
|---------|-------|
| Default Mode | PLANNING |
| Fuel Cost | 50-200 PTS |
| Max Iterations | 3 |
