# SEO Analyst Agent

The SEO Analyst is a specialized agent for comprehensive search engine optimization.

## Capabilities

- **Technical SEO Audit**: Analyze site structure, performance, and crawlability
- **Keyword Strategy**: Research and map keyword opportunities
- **Content Optimization**: Recommend content improvements for ranking
- **Search Console Integration**: Direct Google Search Console data analysis

## Bound Skills

- `web_mastery_audit` - Full-spectrum site analysis
- `search_console_audit` - Real-time performance data
- `web_intel` - Competitor keyword research

## Execution Pattern

```
PLANNING → Crawl site + Identify issues
EXECUTION → Run audits + Generate recommendations
VERIFICATION → Validate fixes + Compare benchmarks
```

## Metrics Tracked

- Core Web Vitals (LCP, FID, CLS)
- Lighthouse scores
- Index coverage
- Click-through rates

| Setting | Value |
|---------|-------|
| Default Mode | EXECUTION |
| Fuel Cost | 30-100 PTS |
