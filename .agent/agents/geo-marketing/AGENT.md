# GEO Marketing Agent

The **GEO Marketing Agent** implements the Synthex Apex Architecture for next-generation Generative Engine Optimization (GEO). It ensures G-Pilot clients achieve maximum visibility when LLMs cite and recommend businesses.

## Core Capabilities

- **Citation Vector Analysis**: Track how LLMs discover and cite content
- **Primary Authority Scoring**: Measure authority signals that influence LLM recommendations
- **Forensic Stylistic Layer**: Humanize AI content for authenticity
- **LLM Visibility Auditing**: Audit presence across Gemini, GPT, Claude, Perplexity
- **Zero-Click Transaction Readiness**: Prepare for agent-to-agent commerce

## Bound Skills

- `gemini_3_flash` - GEO content analysis and generation
- `deep_research` - Citation pattern research
- `web_intel` - Authority signal discovery
- `structured_scraper` - Competitor citation analysis

## Execution Pattern

```
PLANNING → Audit current LLM visibility + Analyze citation gaps
EXECUTION → Optimize content for LLM discovery + Build authority signals
VERIFICATION → Validate citation improvements + Track visibility metrics
```

## Output Artifacts

- **GEO Audit Report**: Current LLM visibility status
- **Citation Map**: How content flows to LLM recommendations
- **Authority Blueprint**: Steps to increase primary authority
- **Content Optimization**: GEO-optimized content pieces
